Módulo 03: Desarrollo Dirigido por Pruebas y Python

Kata Password

Implementar un validador de passwords que compruebe que una cadena de texto:

- Tenga, al menos 6 caracteres
- Tenga, al menos, un _
- Tenga al menos un número
- Tenga, al menos, una letra (mayúscula o minúscula).

Esta solución aún se puede refactorizar más.

El fichero kata_passwd contiene el código implementado durante el hangout del lunes 18/02.
